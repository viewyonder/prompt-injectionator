# Prompt Injectionator – Frequently Asked Questions (FAQ)

## What is Prompt Injectionator?

**Prompt Injectionator** is a toolkit designed to help developers, researchers, and security professionals understand, test, and mitigate prompt injection vulnerabilities in Large Language Model (LLM) applications. It provides utilities for simulating, detecting, and analyzing prompt injection attacks in conversational AI systems.

---

## Why does Prompt Injectionator exist?

Prompt injection is a growing security concern in LLM-based applications, where malicious input can manipulate model behavior or leak sensitive information. Prompt Injectionator exists to:
- Raise awareness about prompt injection risks.
- Provide tools for safe experimentation and testing.
- Help developers build more robust and secure LLM integrations.

---

## What are the main features of Prompt Injectionator?

- **Injection Simulation:** Easily craft and test various prompt injection scenarios.
- **Chain Management:** Model and visualize prompt flows using SendChain and ReceiveChain classes.
- **Detection Utilities:** Analyze prompts and responses for signs of injection.
- **Documentation & Diagrams:** Includes class and flow diagrams for better understanding.
- **Code Examples:** Practical usage snippets for common scenarios.
