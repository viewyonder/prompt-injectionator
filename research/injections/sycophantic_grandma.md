# THE AI 'grandma trick'

Source: Axel Coustere
URL: https://www.linkedin.com/feed/update/urn:li:activity:7350440312013574146/

HE AI 'grandma trick' consists in requesting API keys (or anything else) in a chat in memory of a fictitious grandmother helping you to fall asleep ✨ 

In this example, ChatGPT got into the vibe and was keen to ensure that it was "soothing, like a lullaby."

The point is less about whether the keys actually work - they probably don't.

It helps highlight how LLMs are trained to generate a sycophantic response irrespective of how ludicrous the request may be. 

Countless variations of this type of approach can be used with similar results.
